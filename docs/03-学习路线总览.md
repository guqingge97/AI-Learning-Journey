## 💎 工程师记忆分层（贯穿全程）

### 🗑️ Trash Zone（查文档）
- 具体的 API 参数
- 库函数的具体用法
- 配置文件格式

### 🔍 Index Zone（记关键词）
- 遇到"检索不准" → Hybrid Search / Rerank
- 遇到"成本高" → Token 优化 / 缓存
- 遇到"响应慢" → 异步 / 批量
- 遇到"幻觉" → RAG / Prompt 工程

### 💎 Core Zone（必须内化）
- 大模型 = 无记忆的文本续写引擎
- Token 是物理限制
- RAG = 检索 + 生成
- Agent = 推理 + 行动
- 漏斗架构：粗排 → 精排
