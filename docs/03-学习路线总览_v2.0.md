## 🟣 Month 3: RAG 数据工程 (5周) —— 🔥 核心升级区

> **目标**：精通“喂数据”。引入 **LlamaIndex** 和 **企业级向量库**。

- **Week 9: 向量数据库实战 (🔥 升级)**
  - 原理：Embedding 到底是什么。
  - **工具**：**Milvus (Docker 部署)** 或 Elasticsearch (Vector Search)。
  - *结合点*：用 Java 经验理解 Milvus 的架构。
- **Week 10: LlamaIndex 入门 (🔥 明确)**
  - 核心：`VectorStoreIndex`, `SimpleDirectoryReader`。
  - 实战：用 LlamaIndex 快速建立索引。
- **Week 11: 高级文档处理 (ETL)**
  - 痛点：PDF 表格解析、扫描件 OCR、混乱数据清洗。
  - 工具：LlamaParse 或 Unstructured。
- **Week 12: 切片策略 (Chunking)**
  - 原理：Fixed-size vs Semantic Chunking (语义切分)。
  - 实战：对比不同切分对检索效果的影响。
- **Week 13: 检索策略优化**
  - 技术：Hybrid Search (混合检索 = 关键词 + 向量)、Re-ranking (重排序)。
  - **项目**：构建一个支持混合检索的知识库后端。
