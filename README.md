## 💡 学习方法论

### Teaching Mode（教学协议）

- ✅ 小步慢行，强制互动
- ✅ 不背模板，要理解原理
- ✅ 慢慢思考 > 快速猜答案
- ✅ 每个知识点必须验收

### 全栈宗师笔记体系

每个知识点包含：
- **Why** - 不学会导致的工程死穴
- **What** - 第一性原理 + 类比
- **How** - 最小可运行范式
- **Pitfall** - 真实踩坑
- **Application** - 在哪里用
- **视觉闭环** - 图解理解
- **工程师记忆分层** - 垃圾区/索引区/核心区
